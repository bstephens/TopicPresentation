Overloaded Subprograms

 

 

  • Overloaded subprogram is a subprogram that has the same name as another subprogram in the same referencing environment
  • Every version of an overloaded subprogram must have a unique protocol (e.g. number of parameters, return type, parameter types, etc...)
  • The most common user-defined overloaded methods are constructors

Overloaded subprograms example

void add(int a,int b,int c);
void add(int a,int b);
