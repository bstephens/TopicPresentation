User-Defined Overloaded Operators

 

  • Operators can be overloaded by the user in Adda, C++, Python, and Ruby
  • One example of this is overloading the * operator to compute the dot product of two arrays

Example - assumes Vector_Type has been defined

function "*"(A, B: in Vector_Type) return Integer is
   Sum : Integer := 0;
   begin
   for Index in A'range loop
      Sum := Sum + A(Index) * B(Index);
   end loop;
   return Sum;
   end "*";
